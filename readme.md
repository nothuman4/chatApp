#  Real-Time Chat App (Socket.io + Node.js)

A simple real-time chat application built using **Socket.io**, **Express.js**, and **Node.js**.  
It allows multiple users to join, send, and receive messages instantly — all powered by WebSockets.

---

##  Features
- Real-time messaging (no refresh needed)
- User join/leave notifications
- Simple and clean UI
- Broadcasts messages to all connected users
- Lightweight and fast — perfect for learning Socket.io

---

##  Tech Stack
- **Backend:** Node.js, Express.js, Socket.io  
- **Frontend:** HTML, CSS, JavaScript  
- **Protocol:** WebSocket (via Socket.io)
